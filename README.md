# Mohammed Minhaj Mahmood — Portfolio

Professional single‑page portfolio built with clean HTML/CSS and custom scroll‑spy navigation. Designed to present projects, skills, and contact details in a focused, recruiter‑friendly layout.

## Live
- `https://mohammedminhaj.netlify.app/`

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JS (scroll‑spy for active nav state)

## Features
- Single‑page layout with sticky right‑side nav on desktop
- Sticky bottom nav on mobile
- Scroll‑spy active indicator
- SEO: canonical, Open Graph, Twitter Cards, JSON‑LD, sitemap, robots, webmanifest

## Project Structure
- `index.html` — main page
- `assets/css/style.css` — styling
- `assets/js/scroll.js` — scroll‑spy logic
- `assets/` — images + resume PDF
- `robots.txt`
- `sitemap.xml`
- `site.webmanifest`

## Run Locally
1. Open `index.html` directly in your browser
2. Or serve the folder with any static server

## Customize
- Update text content in `index.html`
- Adjust theme colors in `assets/css/style.css` (`:root` variables)
- Replace the resume PDF in `assets/`

## SEO Notes
If you change the domain, update:
- `index.html` canonical + Open Graph URL
- `robots.txt` sitemap URL
- `sitemap.xml` `<loc>` value
